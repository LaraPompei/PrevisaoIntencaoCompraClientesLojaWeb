# PrevisaoIntencaoCompraClientesLojaWeb
Projeto de machine learning para classificação da intenção de compra de clientes em loja web. Inclui limpeza e imputação de dados, tratamento de inconsistências, análise exploratória, codificação de variáveis categóricas e comparação entre modelos, com Random Forest alcançando F1 médio de 0.8996.
